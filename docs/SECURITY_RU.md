# Безопасность и приватность

Relay постоянно хранит только регистрацию Host: ID, случайные секреты/токены и
последнюю активность. Запросы и ответы находятся в памяти до доставки или
таймаута. Аудио записывается во временный случайный файл на Host и удаляется в
`finally`. Фото могут временно кэшироваться на Host и iPhone с лимитом размера
и срока.

Токен iPhone, секрет Host и админ-ключ Relay являются паролями. После утечки их
нужно заменить. HTTP не шифрует трафик: используйте доверенный VPN либо внешний
TLS reverse proxy. Host слушает только `127.0.0.1`; не открывайте его локальный
порт наружу.
