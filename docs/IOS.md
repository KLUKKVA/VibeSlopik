# iOS client

Install `VibeSlopik.ipa` with your preferred jailbreak/AppSync installer. The
release targets armv7 and iOS 6.0 or newer; iPhone 4s on iOS 6.1.3 is the tested
reference device.

Open **Settings**, enter the complete iPhone URL and token printed by Host, then
save. The Codex tab shows projects and at most ten recent chats. `+` inside a
project starts a chat in that project. A branch can inherit its current model or
use a selected model, supported reasoning effort and approval policy.

Draft text and selected photos are stored per chat. Voice is recorded on the
phone, transcribed on Host and presented for editing before insertion. Images
use thumbnails; disable automatic media loading for slow networks. Tap a loaded
thumbnail for zoom, save and the iOS share sheet. The local media cache is
limited to 50 MiB and can be cleared in Settings.
