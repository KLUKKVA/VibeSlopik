# Частые вопросы

## Нужен ли Mac?

Нет. Готовая IPA устанавливается на jailbreak-устройство через AppSync. Mac и
iOS SDK нужны только разработчику для сборки из исходников.

## Зачем нужен VPS?

Компьютер обычно недоступен из интернета из-за NAT, меняющегося IP и VPN. Relay
даёт постоянную точку встречи, не сохраняя переписку или вложения.

## Можно ли использовать один VPS для VPN и ботов?

Да. Выберите свободный TCP-порт. Установщик не меняет firewall, Docker, VPN и
чужие службы.

## Почему официальный клиент Codex на компьютере обновился не сразу?

VibeSlopik работает через поддерживаемое состояние `app-server`, но не может
заставить другой интерфейс перерисоваться. Перезапустите или обновите официальный
клиент; созданный turn от этого не пропадает.

## Где хранятся фото и голос?

Relay передаёт их только через память. Host создаёт ограниченные временные файлы
и удаляет их по сроку хранения или через очистку кэша. Оригиналы пользователя не
удаляются.

## Что произойдёт после обновления Codex?

Host проверит отдельные возможности протокола. При несовместимости используйте
compatible или diagnostic; forced позволяет попробовать работу без стартовых
проверок и явно помечает функции как непроверенные.
