# Известные ограничения

- Это независимый клиент app-server, а не закрытый OpenAI Remote Control.
- Официальное окно Codex иногда нужно обновить после turn, созданного с iPhone.
- HTTP Relay требует VPN или внешнего TLS для шифрования транспорта.
- Фоновая работа iOS намеренно отсутствует ради батареи.
- В загруженном чате показываются последние 200 нормализованных сообщений, на
  главной — десять недавних чатов; полная история остаётся в Codex.
- iOS 5/7 могут работать, но релизный gate рассчитан на armv7/iOS 6.1.3.
