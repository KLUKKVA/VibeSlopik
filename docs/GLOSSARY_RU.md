# Словарь

- **Codex app-server:** локальный JSON-RPC-процесс Codex, которым управляет Host.
- **Host:** программа на компьютере с доступом к Codex и файлам проектов.
- **Relay:** небольшой посредник на VPS; сам Codex на нём не запускается.
- **Host ID:** открытый идентификатор Host на Relay.
- **Секрет Host:** ключ, с которым Host опрашивает Relay.
- **Токен клиента:** ключ, который вводится в приложении iOS.
- **Админ-ключ Relay:** ключ VPS для регистрации или обновления Host.
- **Ветка/чат:** диалог Codex, связанный с каталогом проекта.
- **Turn:** один запрос пользователя и соответствующая работа Codex.
- **Speech Pack:** необязательный локальный faster-whisper и модель.
- **Режим совместимости:** политика проверок после изменения Codex.
