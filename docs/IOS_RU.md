# iOS-клиент

Установите `VibeSlopik.ipa` удобным jailbreak/AppSync-способом. Релиз собран для
armv7 и iOS 6.0+; основное проверенное устройство — iPhone 4s на iOS 6.1.3.

В **Настройках** введите полный URL для iPhone и токен, показанные Host. Вкладка
Codex показывает проекты и не более десяти недавних чатов. Кнопка `+` внутри
проекта создаёт чат именно в нём. Ветка может наследовать модель либо использовать
выбранную модель, доступный уровень мышления и режим подтверждений.

Текст и выбранные фотографии сохраняются как отдельный черновик каждого чата.
Голос записывается на телефоне, расшифровывается на Host и открывается для
проверки перед вставкой. Фото сначала показываются миниатюрами. Для медленной
сети отключите автозагрузку. Нажмите загруженную миниатюру для масштаба,
сохранения и системного меню. Кэш iPhone ограничен 50 МБ и очищается в настройках.
