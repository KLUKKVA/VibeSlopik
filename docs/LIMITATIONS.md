# Known limitations

- This is an independent app-server client, not OpenAI's private Remote Control.
- An official Codex desktop window may need refresh after a phone-created turn.
- Relay HTTP traffic requires a VPN or external TLS for transport encryption.
- iOS background execution is intentionally absent to protect battery life.
- The client shows the latest 200 normalised messages in a loaded thread and ten
  recent chats on Home; complete history remains in Codex.
- iOS 5 and iOS 7 may work but the release gate targets iOS 6.1.3 on armv7.
