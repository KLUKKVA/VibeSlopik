# Обновление

## Relay

Выполните `sudo vibeslopik-relay update`. Установщик проверит SHA-256 и при
неудачном health check полностью вернёт прежнюю установку. Зарегистрированные
Host остаются в `/opt/vibeslopik-relay/relay-state.json`.

## Desktop Host

Остановите Host, замените распакованную папку новым архивом и снова запустите.
Сохраните каталог `data/`. Host переносит поддерживаемые настройки, а
повреждённую конфигурацию резервирует перед восстановлением defaults.

## iOS

Установите новую `VibeSlopik.ipa` поверх приложения. iOS сохранит настройки и
черновики; перед полным удалением старой версии сделайте резервную копию.

После обновления Codex запустите диагностику Host и изучите статус каждой
функции перед выбором compatible или forced.
