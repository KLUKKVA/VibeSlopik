# VibeSlopik

Удалённое управление Codex с jailbreak-устройств на iOS 6. iPhone работает
только как лёгкий пульт: Codex, проекты, авторизация и необязательное
распознавание речи остаются на компьютере.

[English README](README.md)

## Состав проекта

- **VPS Relay:** небольшой сервис на Go. Передаёт авторизованные запросы через
  память и не хранит чаты, команды, аудио или изображения.
- **Desktop Host:** русско-английская консольная программа для Windows, macOS и
  Linux, подключённая к локальному `codex app-server`.
- **iOS-клиент:** нативное armv7-приложение для iOS 6 и iPhone 4s. Поддерживает
  проекты, чаты, модели, мышление, подтверждения, лимиты, голос, черновики и фото.
- **Speech Pack:** отдельный необязательный пакет. Он явно скачивается по выбору
  пользователя и запускает faster-whisper на компьютере.

## Что требуется

- Jailbreak, iOS 6, armv7 и AppSync для установки IPA.
- Установленный и авторизованный Codex на компьютере.
- VPS с Ubuntu 22.04/24.04 либо Debian 12/13, amd64 или arm64.
- Relay обычно достаточно около 20 МБ RAM. Нагрузка Host в основном зависит от Codex.

## Установка

1. Установите Relay по [инструкции для VPS](docs/VPS_RU.md).
2. Скачайте Host для своей ОС из последнего релиза, распакуйте и запустите
   `VibeSlopik-Host`.
3. Выберите русский язык и пройдите первичную настройку. Введите URL Relay, ID
   Host, секрет Host и админ-ключ, полученные на VPS.
4. Установите `VibeSlopik.ipa`. В настройках приложения введите URL и токен,
   показанные Host.

Программа не меняет firewall автоматически. По умолчанию Relay использует HTTP:
защитите соединение доверенным VPN либо настройте TLS снаружи. Перед публикацией
порта прочитайте раздел [Безопасность](docs/SECURITY_RU.md).
